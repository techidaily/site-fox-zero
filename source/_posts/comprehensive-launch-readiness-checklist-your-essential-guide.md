## Topics

* [System Launch Conditions Tab](https://tools.techidaily.com/advancedinstaller/products/)  
The System Launch Conditions page allows you to select from the conditions already set up in Advanced Installer. These conditions test the OS version and system settings.
* [Software Launch Conditions Tab](https://tools.techidaily.com/advancedinstaller/products/)  
The Software Launch Conditions page allows you to select from the conditions already set up in Advanced Installer. These conditions test if certain software is installed.
* [Custom Launch Conditions Tab](https://tools.techidaily.com/advancedinstaller/products/)  
Allows you to specify your own custom launch conditions.
